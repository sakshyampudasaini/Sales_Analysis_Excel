# Excel-Data-Analytics-Project

## Description
This project demonstrates basic data analytics using Excel. It includes data cleaning, pivot tables, pivot charts, dashboards, and insights.

## Dataset
The dataset contains sales orders with the following columns:
- OrderID
- Customer Name
- Product
- Qty
- Price (USD)
- Order_Date
- Region
- Status
- Notes
- Total Sales
- Order_Month

## Analysis Performed
- Calculated columns: Total Sales, Order Month
- Pivot tables: Sales by Product, Sales by Region
- Pivot Charts / Dashboard: Visual summary of sales data
- Insights: Key observations derived from the data

## Insights
1. Laptop Pro generates the highest revenue: 4802 USD
2. North region has the highest sales: 3693.48 USD
3. USB-C Cable generates the lowest revenue: 199.9 USD

- Microsoft Excel 365
